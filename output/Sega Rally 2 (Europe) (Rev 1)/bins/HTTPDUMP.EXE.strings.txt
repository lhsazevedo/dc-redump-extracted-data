Usage: httpdump <-h host>
                [+p <proxy> | -p]
                [-o <object>]
                [-a <accept types>]
                [-r <referrer>]
                [+ar | -ar]
                [+as | -as]
                [+l <0x??}]
where:
    host         - HTTP server name. i.e. www.microsoft.com.
    +p/-p        - Use/don't use proxy.
                   (Default = Use proxy specified in registry.)
    proxy        - Proxy server name with syntax:
                   [<protocol>=][<scheme>://]<proxy>[:<port>].
    object       - Default object name is default.htm.
    accept types - Data accept ranges. i.e. "text/html".
    referrer     - Referring HTTP link.
    +ar/-ar      - Turn on/off auto-redirect. (Default = on).
    +as/-as      - Turn on/off asynchronous.  (Default = on).
    +s/-s        - Use security (https). (Default = off)
    +l           - Output level: (Default = 0x1111).
                       (Bit 1) Print headers.
                       (Bit 2) Print data.
                       (Bit 3) Print query options.
                       (Bit 4) Print status information.
